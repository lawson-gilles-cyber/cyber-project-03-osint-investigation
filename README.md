# cyber-project-03-osint-investigation
OSINT investigation on a suspicious domain (cyber threat analysis)

## Context

Open Source Intelligence (OSINT) allows analysts to gather information from publicly available sources.

This project simulates an investigation on a suspicious domain.

---

## Objective

* Collect publicly available information
* Analyze domain reputation
* Identify potential threats

---

## 🔍 Target

```id="a9c9cb"

```

## Methodology

1. Domain analysis
2. Reputation check
3. Infrastructure observation

## 🔎 Findings

### 1. Suspicious Domain Name

* Mimics legitimate login services

### 2. No Trusted Reputation

* Not associated with known legitimate services

### 3. Possible Phishing Use

* Likely used for credential harvesting

---

## Conclusion

This domain is likely malicious and potentially used in phishing campaigns.

---

## 🛡️ Recommendations

* Avoid interaction with the domain
* Block the domain in security systems
* Monitor for related activity
